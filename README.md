# vite-react-calculator

Za odabrana dva broja je moguce izvrsiti operacije oduzimanja, sabiranja, mnozenja i dijeljenja.

## Verse link

[https://vite-react-calculator.vercel.app](https://vite-react-calculator.vercel.app)

## Pokretanje lokalno

```bash
git clone https://github.com/filipmilosevic1010/vite-react-calculator.git
cd vite-react-calculator
npm install
npm run dev
```
